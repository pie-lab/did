ICAR Items

Description

ICAR is an acronym for International Cognitive Ability Resource. This file provides a listing of the item labels for the various ICAR items sorted by type. The content needed to administer these items is intentionally excluded here. To access these items, researchers must register at the ICAR-Project website. The items are available in the public-domain but they can only be accessed after registration.
